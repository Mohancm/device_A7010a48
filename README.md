Lenovo K4 Note
==============

The Lenovo K4 Note (codenamed _"A7010a48"_) is a flagship killer smartphone from Lenovo.

This CyanogenMod 13.0 Device tree for K4 Note

For more information about building read "manual"

Basic        | Spec Sheet
-------:     |:-------------------------
